KeyCDN¶

KeyCDN is the leading European content delivery network. The CDN already supports HTTP/2 and consists of 24+ points of presence (POPs) around the world. KeyCDN does not charge extra for HTTPS traffic or HTTP requests. The pricing model is transparent and purely pay-as-you-go. No monthly fee.

KeyCDN Features¶

KeyCDN offers a state-of-the-art features:

  • Full HTTP/2 support for all customers

  • Free Custom SSL based on Let’s Encrypt

  • Raw logs in real time.

  • CDN changes in near-realtime: Any change or update of a KeyCDN zone is globally provisioned within minutes

  • Free WordPress plugins developed by KeyCDN: CDN Enabler and Cache Enabler

  • File browser for push zones: Any content of y push zone can be browsed within the KeyCDN dashboard.

Creating a KeyCDN Push Zone¶

A KeyCDN push zone is needed in order to use Cyberduck and upload content to a zone.

KeyCDN Configuration¶

Once a KeyCDN push zone has been created. Connect with Cyberduck to your push zone. Enter the following information in the bookmark:

  • Protocol: FTP-SSL (Explicit AUTH TLS)

  • Server: ftp.keycdn.com

  • Port: 21

  • Username: The same username as used for the KeyCDN dashboard

  • Password: Same as used for the KeyCDN dashboard

KeyCDN Tools¶

KeyCDN offers a wide variety of tools along with the CDN that can be used:

  • Analyze a website performance with the KeyCDN Speed Test

  • Locate IPs worldwide with IP Location Finder

  • Check if a website is ready for HTTP/2 with the KeyCDN HTTP/2 Test

  • Examine HTTP headers with Curl Test
