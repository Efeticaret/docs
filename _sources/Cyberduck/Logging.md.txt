Logging & Analytics
===

Configuration for the delivery of access logs can be configured from within Cyberduck in the [Info window](Info) for [S3](../Protocols/S3/index), [Google Cloud Storage](../Protocols/Google_Cloud_Storage) and [Amazon CloudFront](../CDN/CloudFront) CDN distributions. The server create a massive amount of log files which is best analyzed with a third-party service.
