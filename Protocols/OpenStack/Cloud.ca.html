Cloud.ca Object Storage¶

Cloud.ca object store is providing cloud storage services with the data located in Canada and powered by renewable energy. The Object store is running on Openstack Swift technology and the infrastructure has been designed for performance and reliability. For additional information please visit Cloud.ca.

Connecting¶

Connection Profile¶

Download the Cloud.ca Object Store East-1 Region Profile for preconfigured settings.

Manual Configuration¶

Enter the following information in the bookmark:

  • Protocol: OpenStack

  • Server: objects.east-1.cloud.ca

  • Port: 443

  • Username: Account Name

  • Password: API Key
