IBM Cloud Object Storage (COS)¶

Attention

  • Softlayer is now IBM Cloud.

  • IBM Bluemix has been rebranded as IBM Cloud within October 2017.

A highly scalable cloud storage service, designed for high durability, resiliency, and security.

Connecting¶

Credentials¶

Obtain HMAC credentials.

Users can create a set of HMAC credentials as part of a Service Credential with the use of the advanced configuration parameter {“HMAC”:true} during credential creation in the console.

Connection Profiles¶

Using Cross Region Endpoints

  • us-geo: Download the IBM COS (us-geo) Connection Profile for preconfigured settings.

  • eu-geo: Download the IBM COS (eu-geo) Connection Profile for preconfigured settings.

  • ap-geo: Download the IBM COS (ap-geo) Connection Profile for preconfigured settings.
