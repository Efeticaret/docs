Verizon Cloud Storage¶

Verizon Cloud Storage. Verizon Cloud Storage is an object storage service compatible with S3 API.

Connecting¶

You create storage keys from the console at https://my.cloud.verizon.com/profile/#/api-keys.

Connection Profiles¶

  • Verizon Cloud Storage (East Coast USA): Download the Verizon Cloud Storage (IAD3A) Connection Profile for preconfigured settings.

  • Verizon Cloud Storage (Netherlands, EU): Download the Verizon Cloud Storage (AMS1A) Connection Profile for preconfigured settings.

  • Verizon Cloud Storage (3 Data Center Spread, USA): Download the Verizon Cloud Storage (USHAA) Connection Profile for preconfigured settings.

Manual Configuration¶

Enter the following information in the bookmark:

  • Protocol: S3 (Amazon Simple Storage Service)

  • Server: storage-iad3a.cloud.verizon.com, storage-ams1a.cloud.verizon.com, storage-ushaa.cloud.verizon.com
