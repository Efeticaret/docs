MinIO Cloud Storage

MinIO is an object storage server built for cloud application developers and DevOps.
Store photos, videos, VMs, containers, log files, or any blob of data as objects.

Connecting

Download and install the generic S3 profile.

Virtual Host Style Requests

Make sure to enable virtual-host-style requests in your MinIO configuration. Alternatively refer to Disable use of Virtual Host Style Requests.
